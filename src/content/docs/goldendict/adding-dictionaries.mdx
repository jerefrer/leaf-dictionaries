---

- Add your new dictionary files in the same folder you used above

- Return to `Edit > Dictionaries` or press `F3`

- Click `Rescan now` to index only the new dictionaries. The existing ones won't be indexed again.

- Wait a little for the indexing to be complete.

- That's it!
